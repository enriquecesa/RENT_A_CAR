Es un campo dinámico que combina creatividad, lógica y una comprensión profunda de cómo los usuarios interactúan con la información en línea. Se divide principalmente en dos grandes áreas: el desarrollo frontend y el desarrollo backend. ¿Qué es el Desarrollo Web? El desarrollo web es el proceso de crear sitios y aplicaciones para Internet o una intranet. Incluye tareas como el diseño web, el desarrollo de contenido web, la programación de scripts del lado del cliente/servidor y la seguridad de la red. En esencia, es todo lo que hace que un sitio web funcione, desde su apariencia visual hasta su lógica interna y la forma en que interactúa con las bases de datos. ¿Qué es el Diseño Web? El diseño web se centra en la parte visual y la experiencia de usuario de un sitio web. Va más allá de lo estético; busca crear interfaces que sean atractivas, intuitivas y fáciles de usar. Los diseñadores web se preocupan por la disposición de los elementos (layout), la combinación de colores, la tipografía, las imágenes y la forma en que los usuarios navegarán por el sitio. Su objetivo es asegurar que la interacción del usuario sea placentera y eficiente. ¿Qué es el Maquetado Web? El maquetado web es la fase en la que se traduce el diseño visual (creado por el diseñador web) a código. Es el proceso de estructurar el contenido de una página web utilizando lenguajes como HTML y CSS. Un maquetador se encarga de posicionar textos, imágenes, videos y otros elementos de acuerdo con el diseño, asegurando que el sitio sea responsive (se adapte a diferentes tamaños de pantalla) y accesible. Es el puente entre el diseño y la programación. ¿Qué es la Programación Web? La programación web se refiere a la creación de la lógica y la funcionalidad que permiten a un sitio web interactuar y realizar tareas. Esto puede incluir desde animaciones y validaciones de formularios en el navegador (frontend) hasta el manejo de bases de datos, la autenticación de usuarios y la lógica de negocio en el servidor (backend). Se utilizan lenguajes de programación para dar vida a la interacción y al procesamiento de datos. Tecnologías Web Fundamentales Para construir una página web como la que acabamos de crear, se utilizan varias tecnologías clave. A continuación, se explican las más básicas y su evolución histórica: 1. HTML (HyperText Markup Language) ¿Qué es? HTML es el lenguaje estándar para crear y estructurar el contenido de las páginas web. No es un lenguaje de programación, sino un "lenguaje de marcado" que utiliza etiquetas para definir la estructura de la página (títulos, párrafos, imágenes, enlaces, listas, etc.). Explicación: Imagina HTML como el esqueleto de tu página web. Define dónde van a estar cada uno de los elementos de contenido. Orden Histórico: Fue creado por Tim Berners-Lee a principios de los 90. HTML5 es la última y más avanzada versión, lanzada en 2014, que introdujo nuevas etiquetas semánticas (

), soporte para video y audio nativos, y APIs para interacciones más complejas, mejorando significativamente la estructura y la accesibilidad. 2. CSS (Cascading Style Sheets) ¿Qué es? CSS es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en HTML. Permite controlar la apariencia de las páginas web, como colores, fuentes, espaciado, diseño (layout) y animaciones. Explicación: Si HTML es el esqueleto, CSS es la piel, la ropa y el maquillaje. Le da estilo y embellece el contenido estructurado por HTML. Orden Histórico: Introducido a mediados de los 90, CSS3 (lanzado gradualmente a partir de finales de los 90 y principios de los 2000) añadió características muy potentes como bordes redondeados, sombras, transformaciones 2D/3D, transiciones, animaciones y diseño responsive (@media queries), permitiendo diseños mucho más complejos y dinámicos sin necesidad de JavaScript adicional. 3. Bootstrap ¿Qué es? Bootstrap es un framework de CSS de código abierto, diseñado para facilitar la creación de sitios web responsive y mobile-first (primero para móvil). Incluye plantillas de diseño basadas en HTML y CSS para tipografía, formularios, botones, navegación y otros componentes de interfaz, así como extensiones de JavaScript. Explicación: Bootstrap es como tener un kit de construcción prefabricado para tu sitio web. Te proporciona componentes ya estilizados y un sistema de cuadrícula que te ahorra mucho tiempo al diseñar layouts responsivos. Orden Histórico: Lanzado originalmente por Twitter en 2011. Bootstrap 5.3.3 (como el usado en este proyecto) es una versión reciente que ha eliminado la dependencia de jQuery, mejorado el soporte de CSS variables, y optimizado la personalización, haciéndolo más moderno y ligero. 4. Font Awesome ¿Qué es? Font Awesome es una popular biblioteca de iconos vectoriales escalables que se puede personalizar fácilmente con CSS. Permite insertar iconos web en el diseño, como los que representan un usuario, un sobre o un proyecto. Explicación: Es una fuente especial que, en lugar de letras, contiene miles de iconos. Puedes cambiar su tamaño y color como si fueran texto normal. Orden Histórico: Se hizo popular a principios de los 2010. Font Awesome 6.5.1 es una de las últimas versiones, con una biblioteca de iconos mucho más extensa y opciones de estilo mejoradas. 5. JavaScript (JS) ¿Qué es? JavaScript es un lenguaje de programación que permite añadir interactividad a las páginas web. Se ejecuta directamente en el navegador del usuario (cliente-side) y puede manipular el HTML y CSS de la página, responder a eventos del usuario, realizar validaciones y comunicarse con servidores. Explicación: Si HTML es la estructura y CSS es el estilo, JavaScript es el cerebro y los músculos que hacen que la página cobre vida y reaccione a lo que haces. Orden Histórico: Creado en 1995 por Brendan Eich en Netscape. Desde entonces, ha evolucionado de ser un lenguaje simple para scripts de navegador a un lenguaje robusto capaz de desarrollar aplicaciones complejas tanto en el frontend como en el backend (con Node.js). El término "JavaScript vanilla" se refiere al uso puro de JavaScript, sin frameworks o bibliotecas adicionales como React, Angular o Vue, que son más recientes.
